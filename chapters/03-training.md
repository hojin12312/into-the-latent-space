# 03. 학습: 도서관을 통째로 읽는 법

<!-- TODO: 본문 작성 -->

**핵심 비유**: 거대한 도서관 통째로 읽기 + 빈칸 채우기 시험 수억 번

**도입부 훅**: "AI 가 똑똑해지는 데 든 전기료가 작은 도시 한 달 사용량보다 많을 수 있어요" 같은 숫자 충격으로 시작

**다룰 핵심 개념**: 사전학습(pretraining), 파라미터(parameter)란 무엇인가, 데이터셋의 규모, GPU 클러스터

**깜짝 사실 후보**:
- 학습이라는 게 사실 "틀렸을 때마다 살짝씩 수치를 조정" 하는 단순한 반복이라는 점
- 파라미터가 사람 뇌의 시냅스 개수와 비교했을 때의 감각

**다이어그램 후보**:
- 빈칸 채우기 시험 사이클: `[빈칸 있는 문장] → [모델 추측] → [정답과 비교] → [살짝 조정]` → 반복 (Mermaid 흐름도가 잘 어울림)
- 모델 크기 비교 막대그래프 (ASCII): GPT-2 → GPT-3 → GPT-4 의 파라미터 수 단계적 증가
