# 04. 답이 나오는 순간

<!-- TODO: 본문 작성 -->

**핵심 비유**: 한 글자씩 던지는 룰렛 (온도 = 룰렛판 흔드는 세기)

**도입부 훅**: "AI 한테 같은 질문을 두 번 하면 다른 답이 나오는 이유 — 사실 룰렛을 돌리고 있어서예요"

**다룰 핵심 개념**: 추론(inference), 온도(temperature), 샘플링(sampling), 확률 분포에서 단어 뽑기

**깜짝 사실 후보**:
- temperature = 0 이면 항상 같은 답이 나오지만 인간이 보기엔 더 단조롭다는 것
- 답을 만들면서 AI 자신은 자기가 뭘 쓰는지 한 단어 앞만 보고 있다는 것

**참고 자산**: `../../llm-field-notes/appendix/theory/transformer-inference.md` — prefill / decode 같은 단어는 본문에서 빼고 비유로만.

**다이어그램 후보**:
- 다음 단어 후보들의 확률 분포 ASCII 막대그래프 (예: "오늘 날씨가" 다음에 올 후보로 "좋다 ████ / 흐리다 ██ / 더럽다 ▏ ..." 식)
- 온도(temperature) 비교: T=0 (항상 1등 뽑기), T=0.7 (적당히 흔들기), T=1.5 (난장판) — 같은 입력에서 나오는 출력 차이
